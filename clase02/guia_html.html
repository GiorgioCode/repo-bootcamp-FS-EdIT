Estructura Básica de un Documento HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título del Documento</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

Etiquetas Semánticas

Artículo de Ejemplo

Un <article> es contenido independiente y auto-contenido.

Barra lateral

<aside> se usa para contenido relacionado pero no esencial.

Subsección

<section> agrupa contenido temáticamente relacionado.

Metadatos

Los metadatos se colocan dentro de <head> y proveen información sobre el documento.

  • <meta charset="UTF-8">: Codificación de caracteres
  • <meta name="viewport">: Adaptación a móviles
  • <title>: Título que aparece en la pestaña
